# 📝 Posting Page

Projeto desenvolvido como **desafio de certificação do curso DevStart**, com o objetivo de praticar **HTML, CSS e JavaScript**, além do consumo de **APIs utilizando Fetch**.

A aplicação permite que o usuário crie um post com **título e conteúdo**, enviando esses dados para uma API e exibindo o resultado na própria página.

---

## 🚀 Funcionalidades

* Criar um post com **título e conteúdo**
* Enviar os dados para uma **API externa**
* Exibir o post criado na tela
* Limpar o formulário após o envio
* Interface simples e responsiva

---

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **Fetch API**
* **JSONPlaceholder API**

A API utilizada para simulação de postagem foi:

🔗 [https://jsonplaceholder.typicode.com/](https://jsonplaceholder.typicode.com/)

Essa API é utilizada para **testes e protótipos**, simulando requisições HTTP reais.

---

## 📂 Estrutura do projeto

```
posting-page
```

